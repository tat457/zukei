# うしのさくゲーム

三頭の牛をゆっくり誘導しながら、指定された図形の形に近い柵をひと筆で描いて囲むブラウザゲームです。  
ステージは `円 / 三角 / 四角 / 台形 / 五角形` の5種類です。
スタートを押すと三頭の牛が画面の中をゆっくり動きます。  
プレイヤーは `円 / 三角 / 四角 / 台形 / 五角形` から図形を選び、その形に近い柵を描いて牛を3頭とも囲めたらクリアになるブラウザゲームです。

## 遊び方

1. `index.html` をブラウザで開きます。
2. `はじめる` を押します。
3. マウスドラッグ、またはスマホならタッチ操作で牛たちを動かします。
4. 図形の外側を囲うように進み、スタート地点の近くに戻ると判定されます。
2. 図形を選びます。
3. `はじめる` を押します。
4. マウスドラッグ、またはスマホならタッチ操作で柵を描きます。
5. 指やマウスを離したときに柵が閉じ、牛3頭を囲えていて選択図形に近ければクリアです。
