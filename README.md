# うしのさくゲーム

スタートを押すと三頭の牛が画面の中をゆっくり動きます。  
プレイヤーは `円 / 三角 / 四角 / 台形 / 五角形` から図形を選び、その形に近い柵を描いて牛を3頭とも囲めたらクリアになるブラウザゲームです。
最初にタイトル画面でゲーム名、スタートボタン、図形選択ボタンが表示されます。  
スタートを押すとゲーム画面へ切り替わり、三頭の牛が画面の中をゆっくり動きます。プレイヤーは選んだ `円 / 三角 / 四角 / 台形 / 五角形` の形に近い柵を描いて牛を3頭とも囲めたらクリアです。

## 遊び方

1. `index.html` をブラウザで開きます。
2. 図形を選びます。
3. `はじめる` を押します。
2. タイトル画面で図形を選びます。
3. `スタート` を押してゲーム画面へ進みます。
4. マウスドラッグ、またはスマホならタッチ操作で柵を描きます。
5. 指やマウスを離したときに柵が閉じ、牛3頭を囲えていて選択図形に近ければクリアです。
