# うしのさくゲーム

三頭の牛をゆっくり誘導しながら、指定された図形の形に近い柵をひと筆で描いて囲むブラウザゲームです。  
ステージは `円 / 三角 / 四角 / 台形 / 五角形` の5種類です。

## 遊び方

1. `index.html` をブラウザで開きます。
2. `はじめる` を押します。
3. マウスドラッグ、またはスマホならタッチ操作で牛たちを動かします。
4. 図形の外側を囲うように進み、スタート地点の近くに戻ると判定されます。

## ファイル構成

- `index.html` : 画面全体のレイアウト
- `style.css` : 見た目とレスポンシブ対応
- `script.js` : ゲームロジック、描画、判定

## GitHub で公開する方法

静的ファイルだけで動くので、GitHub Pages にそのまま置けます。

1. このフォルダを GitHub リポジトリに push します。
2. GitHub の `Settings > Pages` を開きます。
3. `Deploy from a branch` を選び、`main` ブランチの `/root` を公開対象にします。
4. 数十秒待つと公開URLで遊べます。
