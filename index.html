Warm-up Shape Game

うしのさくゲーム

ゆっくり動く三頭の牛をうまく誘導して、指定された図形をひと筆の柵で囲もう。 図形に近い形で閉じられたらクリアです。 スタートを押すと、画面の上を三頭の牛がゆっくり動き出します。 選んだ図形の形で牛たち全員をひとつの柵で囲めたらクリアです。

ステージ 選択図形 円
クリア数 0 / 5 成功回数 0
操作 ドラッグ / タッチ 押して描く
牛たちをドラッグで導こう
スタート地点へ戻ると柵が閉じます
牛が3頭そろったら囲もう
離すと柵が閉じて判定されます

あそびかた

  1. 「はじめる」を押すと、スタート地点から柵づくりが始まります。
  2. 牛たちはゆっくり追いかけてくるので、図形の外側をなぞるように動かします。
  3. スタート地点の近くへ戻ると柵が閉じて判定されます。
  4. まず囲いたい図形を選びます。
  5. 「はじめる」を押すと三頭の牛が動き始めます。
  6. マウスやタッチで柵を描き、牛を3頭とも囲んだまま離すと判定されます。
